# デジみみぃ
周囲のWi-Fiアクセスポイントを検出して表示します

# 利用ハード
- M5Stick S3

# 開発環境
- 統合開発環境: Arduino IDE
- ボードマネージャ: M5Stick by M5Stack Offcial Ver3.2.6(3.3.7だとWifi関連が動作しない)

# パラメータと音の関係
- 固有ID(bssid): 楽器種別
- 電波強度(rssi): 音量 
- チャンネル(channel): 音階

# 音階
 ペンタトニックスケール（Cメジャー）を利用
## 理由
- どの音を同時に鳴らしても比較的きれい
- Wi-Fiが10台以上あっても耳障りになりにくい
- 「環境音楽」「アンビエント」に向いている
